mlxmolkit
=========

GPU-accelerated molecular 3D coordinate generation on Apple Silicon using MLX.

mlxmolkit is a drop-in replacement for RDKit's ETKDG conformer generation that
runs on Apple Silicon GPUs via the `MLX <https://github.com/ml-explore/mlx>`_
framework.
