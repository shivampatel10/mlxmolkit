Installation
============

Requirements
------------

- macOS with Apple Silicon (M1/M2/M3/M4)
- Python >= 3.12

Install from PyPI
-----------------

.. code-block:: bash

   pip install mlxmolkit

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/shivampatel10/mlxmolkit.git
   cd mlxmolkit
   pip install -e .
